# Contributing to Awesome Agent Skills

Thanks for your interest in contributing! This project aims to be the most comprehensive cross-platform directory of AI agent skills.

## How to Contribute

### Adding a new skill/resource

1. Fork this repository
2. Add your entry to the appropriate section in `README.md`
3. Use this format:
   ```
   - [name](url) - Short description. ![GitHub stars](https://img.shields.io/github/stars/owner/repo)
   ```
4. Keep descriptions concise (under 120 characters)
5. Submit a pull request

### Guidelines

- **One entry per PR** (makes review faster)
- **No duplicates** — search the list first
- **Working links only** — we check automatically
- **Relevant content** — must be a skill, tool, plugin, rule, or extension for an AI agent
- **Quality bar** — should have at least 10 GitHub stars or be from an official source
- **Alphabetical order** within sections (after official/featured entries)

### What we accept

- Agent Skills (SKILL.md format)
- MCP servers and tools
- Cursor rules and rule collections
- Windsurf rules
- Gemini CLI extensions
- GitHub Copilot extensions
- OpenClaw skills
- LangChain/CrewAI/framework-specific tools
- CLI tools for skill management
- Directories and marketplaces
- Tutorials and guides about creating skills

### What we don't accept

- Paid products without free tiers
- Abandoned projects (no commits in 6+ months, unless historically significant)
- Duplicate entries
- Self-promotional spam
- Projects unrelated to AI agent skills

## Automated Sync

Some sections are automatically updated via our sync pipeline. If you notice outdated data in synced sections, please open an issue rather than a PR — we'll fix the sync.

## Code of Conduct

Be kind. Be helpful. We're all here to make AI agents more capable.
